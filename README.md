# Projektanforderungen

## Übersicht

Dieses Projekt ist eine Anwendung zur Verwaltung von Waren, die von Kunden in einem Unternehmen abgegeben werden. Es beinhaltet Benutzerrollen, Warenverwaltung, Rechnungen und Vertragsverwaltung.

## Benutzerrollen

**Admin**

- Kann neue Benutzer erstellen
- Kann Preise für Waren ändern
- Hat Zugriff auf Abrechnungen
- Kann Verträge einsehen

**Normaler Benutzer**

- Eingeschränkte Rechte
- Kann keine Preise ändern oder neue Benutzer erstellen
- Kann Verträge einsehen

## Funktionen

### Kunden und Warenverwaltung

- Kunden können Waren abgeben und erhalten eine Rechnung
- Das System speichert eingegangene und ausgegangene Waren
- Eine Übersicht über den Warenbestand wird bereitgestellt

### Rechnungen

- Beim Eintreffen eines Kunden wird eine Rechnung erstellt
- Der Benutzer gibt die Art und Anzahl der abgegebenen Waren ein
- Die Rechnung kann ausgedruckt und vom Kunden unterschrieben werden

### Übersicht

- Darstellung aller eingegangenen und ausgegangenen Waren
- Mengenangaben in Tonnen oder Kilogramm

### Verträge

- Erstellung von Verträgen für das Einlagern und Abholen von Waren

### Login-System

- Rollenbasiertes Login mit Admin- und Benutzerrechten
